## ⚡ Quick Start (Versi Cepat)

```bash
# Clone repositori
git clone https://github.com/Nothingness1312/RCE-Lab-Platform.git

# Masuk ke direktori
cd RCE-Lab-Platform

# Setup environment
cp .env.example .env

# Jalankan dengan Docker
docker compose up -d
